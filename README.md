# Country Explorer

## Project Description
Country Explorer is a web application that allows users to search and explore information about countries around the world.

Users can:
- Search for a country by name
- View country details such as population, capital, region, and flag
- Explore real-time data retrieved from an external API
- Easily navigate and interact with a clean and responsive interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- External REST API

---

## External API Used

This project uses the **REST Countries API** to retrieve country data:

https://restcountries.com/

All country information is fetched dynamically using JavaScript (fetch + JSON).

---

## How to Run the Project

1. Extract the ZIP file.
2. Open the folder.
3. Double-click on `index.html`.
4. The application will open in your default browser.

 Make sure all files remain in their original structure:
COUNTRYEXPLORER/
index.html
css/
style.css
js/
script.js
README.md
